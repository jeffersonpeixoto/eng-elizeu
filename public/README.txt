CONFIGURAÇÃO

1. Abra o arquivo supabase.js
2. Preencha:
   const SUPABASE_URL = "https://SEU-PROJETO.supabase.co";
   const SUPABASE_KEY = "SUA_ANON_KEY";

3. Execute em servidor local:
   python -m http.server 8000

4. Abra no navegador:
   http://localhost:8000
